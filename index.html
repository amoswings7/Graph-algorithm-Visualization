What is a Graph?

A graph in computer science or software engineering is just a representation of data that we can use and work with. We can use it to check if two points are connected or if there is relationship between them. an example would be if there is a path between New York City and Florida.

We can also use a graph to search for a particular data, there are mainly two searching algorithms or searching methods we can use. One of them is called Breadth first search and the other is called Depth first search. They both got their strengths and weaknesses and you can check them out below of how they work
